# Clone Tracker
Real-time clone tracker with neon effects, trails, background blur, gesture detection, and recording.

## Run
1. Create venv: `py -3.10 -m venv .venv`
2. Activate: `.\.venv\Scripts\Activate.ps1`
3. Install: `pip install -r requirements.txt`
4. Run: `python clone_tracker_full_tk.py`

## Files
- `clone_tracker_full_tk.py` — main app
- `requirements.txt` — dependencies
